
# Inglês na Barca — Full Deployable Repository

Conteúdo:
- Ocean map (`index.html`)
- Unit navigation (`unit.html`)
- Lesson engine (`lesson.html`)
- Styles (`styles.css`)
- App logic (`app.js`)
- Course data (`course-data.js`)
- 120 lesson JSON files in `/lessons`

Publicação:
1. Envie todos os arquivos para a raiz do repositório.
2. Preserve a pasta `lessons`.
3. Ative o GitHub Pages em Settings > Pages.
4. Abra o site publicado.
